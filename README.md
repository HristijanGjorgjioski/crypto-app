## About The Project

![Screen Shot](https://imgur.com/v4SdAod.png)

This is web application that can be used like Crypto news and Crypto info site. In the application there is Home page where are listed top 12 cryptos coins
and some info about them. In Cryptocurrencies page you can find top 100 cryptos coin, and by clicking on them you are redirected to crypto details page where
you can find everything about that specific coin. There is also Exchange and News page where you can find all crypto exchanges and read info about them
and in the News page there is a lot of news about crypto.
